In order to build this plugin you will need;

1. Berkelium binaries (11.0.696.57), download here: https://github.com/sirikata/berkelium/downloads
2. Awesomium SDK (1.6.4), download here: http://awesomium.com/download/
3. PixelLight, download here: http://www.pixellight.org/site/index.php/page/2.html

Do not forget to include these files in order to run this plugin;

* berkelium.exe
* berkelium.dll
* icudt46.dll
* resources.pak
* wow_helper.exe
* locales\en-US.dll

And for Awesomium

* Awesomium.dll
* AwesomiumProcess (optional)
* AwesomiumProcess.exe
* icudt42.dll
* avcodec-52.dll
* avformat-52.dll
* avutil-50.dll
* locales\en-US.dll

These files are included with the Berkelium and Awesomium binaries.

NOTE: If flash does not work in Awesomium try the following things;
1. Download and install Adobe Flash Player from, ..
2. Make sure you have NPSWF32.dll in your "C:\Windows\System32\Macromed\Flash\" or for Windows 64 bit "C:\Windows\SysWOW64\Macromed\Flash\"
3. Still does not work? copy NPSWF32.dll to your executable directory
